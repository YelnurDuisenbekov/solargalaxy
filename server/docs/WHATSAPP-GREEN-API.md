# WhatsApp через Green API (без Facebook)

Meta часто **не даёт создать приложение** — это нормально. Green API работает через QR-код вашего номера.

**Ваш номер:** +7 777 475 1332

## За 5 минут

### 1. Регистрация
Откройте [console.green-api.com](https://console.green-api.com) → регистрация (есть бесплатный тариф).

### 2. QR-код
- Создайте **инстанс**
- На телефоне с WhatsApp **+7 777 475 1332**:
  - WhatsApp → **Настройки** → **Связанные устройства** → **Привязка**
  - Отсканируйте QR из личного кабинета Green API

### 3. Ключи
На странице инстанса скопируйте:
- **idInstance** (например `1101123456`)
- **apiTokenInstance** (длинная строка)

### 4. Автонастройка проекта

Двойной клик:
```
server/setup-green-api.bat
```

Или в терминале:
```bash
npm run green:setup
```

Вставьте idInstance и apiToken — скрипт проверит QR и запишет `.env`.

### 5. Готово
```bash
npm run dev
```
CRM → **Система → WhatsApp API** → статус «Работает».

## Что получите
- Кнопка **WhatsApp API** в CRM — сообщение уходит автоматически
- **Follow-up через сутки** после «Заинтересован» — без шаблонов Meta
- **Напоминания менеджерам** в 10:00 — как и раньше
